
# Content Intelligence System · v5

A Python-based content intelligence system that analyzes historical content performance data, generates benchmark models, and provides data-driven recommendations to support content optimization decisions.

## Background

This project was inspired by a real-world content management and digital marketing workflow.

The original business challenge involved evaluating large volumes of published content and identifying patterns associated with higher engagement and stronger SEO performance.

To demonstrate the underlying technical approach without exposing proprietary information, this repository uses a synthetic dataset that replicates the structure and characteristics of real-world content analytics data.

---
## Data & Privacy Notice

This repository does not contain any proprietary, confidential, or personally identifiable information.

The dataset included in this project is a synthetic dataset created solely for demonstration and educational purposes. The structure of the data is inspired by real-world content analytics workflows, but all records, metrics, and article information have been anonymized or generated independently.

This project demonstrates the technical approach, system design, and analytical methodology rather than any specific organization's data.

## Technical Highlights

- Python-based analytics pipeline
- Benchmark generation from historical datasets
- Weighted scoring model for content evaluation
- Rule-based recommendation engine
- Opportunity detection (high SEO / low visibility content)
- Gap analysis for content planning
- Monthly content planner with per-topic historical analysis
- Excel report generation with embedded charts for marketing managers
- Modular script architecture (scoring, planner, reporting)
- Flask backend serving the tool and handling AI API calls
- Provider-agnostic AI suggestions (Gemini, OpenAI, or Anthropic — swap in one line)
- HTML-based evaluation interface with tabbed navigation
- JSON benchmark and planner data generation and consumption

## How it works

### Value Score (0–100)

Each post is scored on four marketing metrics, each percentile-ranked
against the full dataset so the score reflects *relative* performance:

| Metric                        | Weight |
|-------------------------------|--------|
| View Count                    | 30 %   |
| SEO Linkdex (focus keyword)   | 25 %   |
| SEO Content Score             | 25 %   |
| Estimated Reading Time        | 20 %   |

Reading time is scored against an ideal engagement band (2–4 min)
rather than raw percentile — more is not always better.

### Value Tiers

| Tier        | Score  | Meaning                                       |
|-------------|--------|-----------------------------------------------|
| Gold        | 80–100 | Top performer — study and replicate           |
| Silver      | 60–79  | Solid — small improvements push to Gold       |
| Bronze      | 40–59  | Needs work — review SEO and content depth     |
| Needs Work  | 0–39   | Underperformer — prioritise SEO or topic rethink |

### Topic Categories (inferred from titles)

- Aquatics & Swim School
- Equipment Upgrades
- Facility Operations
- Group Fitness
- Health & Wellness
- Programs & Enrolments
- Trainer Spotlight

## Quick-start

### 1. Install dependencies

```bash
pip3 install -r requirements.txt
```

### 2. Set up your API key

```bash
cp .env.example .env
# Open .env and add your API key for your chosen provider
```

The default provider is **Gemini** (free tier available at https://aistudio.google.com).
To switch providers, update `PROVIDER` in `ai/suggest.py` — supports Gemini, OpenAI, and Anthropic.

### 3. Run all scripts

Run in order — or whenever you add new posts to the CSV:

```bash
python3 scoring/build_benchmarks.py
python3 scoring/analyse_opportunities.py
python3 planner/generate_planner.py
python3 reporting/export_report.py
```

### 4. Start the Flask server

```bash
python3 app.py
# open http://127.0.0.1:5000
```

> **Note:** v5 uses Flask instead of `python3 -m http.server` — the Flask server handles AI suggestions securely server-side. Always use `python3 app.py` to run the tool.

### 5. Use the tool

The tool has two tabs:

**Post Checker** — paste a draft post title, select a category, paste content body. The tool will:
1. Estimate reading time from word count
2. Project a Value Score against category benchmarks
3. Show a Tier rating (Gold / Silver / Bronze / Needs Work)
4. Show actionable guidance based on your score
5. Show Post Brief and top historical posts to benchmark against
6. Click **✨ Get AI Suggestions** for AI-powered title rewrites and content improvements

**What to Write** — pick a month to see:
1. Post ideas based on historically strong topics for that month
2. Topic coverage map showing missing and underrepresented categories
3. Top historical posts from that month for reference

## What each script produces

| Script | Output | Contains |
|---|---|---|
| `scoring/build_benchmarks.py` | `benchmarks.json` | Per-topic benchmarks, brief templates, guidance rules |
| `scoring/analyse_opportunities.py` | `opportunities.json` | Undervalued posts, overexposed posts, gap analysis |
| `planner/generate_planner.py` | `planner_data.json` | Per-month topic performance and post ideas |
| `reporting/export_report.py` | `content_report.xlsx` | 4-sheet Excel report with embedded charts |
| `ai/suggest.py` | API response | AI title and content suggestions (Gemini / OpenAI / Anthropic) |
| `app.py` | Flask server | Serves the tool at http://127.0.0.1:5000 |

### Excel Report — content_report.xlsx

Generated by `export_report.py` for the marketing manager. Open directly in Excel or Numbers — no code required.

| Sheet | Contents |
|---|---|
| **Summary** | Key metrics, tier breakdown chart, top 3 insights |
| **What's Working** | Top 10 posts by value score, topic performance chart |
| **Quick Wins** | Undervalued posts to promote + overexposed posts to fix |
| **Content Gaps** | Monthly volume chart + underperforming categories |

### Post Brief Template (per category)

Each category in `benchmarks.json` includes a ready-to-use brief:

```json
"brief_template": {
  "ideal_reading_time": { "min": 2, "max": 4 },
  "target_seo_linkdex": 70.0,
  "target_seo_content": 75.0,
  "min_value_score_before_publish": 52.3,
  "winning_title_formats": [
    "Why You Should Add [X] to Your Weekly Routine",
    "The [Adjective] Health Benefits of [X]"
  ]
}
```

## Opportunities identified in current dataset

Run `build_benchmarks.py` and open `benchmarks.json` to see the full lists.
High-level findings from 300 posts (2023–2026):

- **Undervalued posts (10):** High SEO scores but low view counts.
  Action: promote via social media and internal linking.
- **Overexposed posts (10):** High views but weak SEO foundations.
  Action: refresh meta descriptions, headings, and keyword use.
- **Seasonal gap:** September has below-average post volume.
  Plan 1–2 extra posts in that month.

## Refreshing the data

1. Export a new CSV from your CMS in the same column format.
2. Place it in the `data/` folder (same filename).
3. Re-run all scripts:
```bash
python3 scoring/build_benchmarks.py
python3 scoring/analyse_opportunities.py
python3 planner/generate_planner.py
python3 reporting/export_report.py
```
4. Restart Flask: `python3 app.py`
5. Hard refresh the browser — **Cmd + Shift + R**.
6. Open the new `data/content_report.xlsx` to share with the marketing manager.
